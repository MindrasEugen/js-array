# Esercizio

Dato un array di prodotti e uno di quantità, stampa se il prodotto è esaurito o se le scorte sono basse.

## ragionamento

- dichiaro contenitore prodotti
- dichiaro contenitore quantita
- PER ogni quantita
    - SE la quantita = 0
        - stampo prodotto esaurito
    - ALTRIMENTI 
        - SE la quantita e < 5
            - stampo le scorte sono base
