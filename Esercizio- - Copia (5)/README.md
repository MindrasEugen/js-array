# esercizio

Data la lista VIP per una serata, verifica se l'utente X è presente nella lista VIP o se è nella lista standard

## ragionamento

- definisco la lista VIP
- dichiaro l'utente X
- SE listaVip.includes(utenteX)
    - stampo l'utente x e nella lista vip
- ALTRIMENTI
    - stampo l'utente e nella lista standard
